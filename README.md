# 당근게임
벌레를 피해 당근 캐기 게임입니다 ^^


### 게임시작
[당근게임 하러가기!](https://kah9509.github.io/Carrot-Game/)

### 게임방법
그저...벌레말고 당근 클릭해주세요.

### 기술스택
이 프로젝트에 사용된 기술
* Language: Javascript

### 기능 & 목표
- 게임 시작/재시작 시 시간 초기화 후 진행
- 순수 자바스크립트로 짜보기 (자바스크립트 이해를 위해)
- 코드 리펙토링
  - 일반적인 반복문 map으로 변경
  - 클래스화
- 소소한 재미..를 위해
